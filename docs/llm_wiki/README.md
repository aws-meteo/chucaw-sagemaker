## 2. Correct Mental Model
* **Batch-First Inference Model:** We only create SageMaker Model metadata records. We **never** spin up active compute, real-time endpoints, or endpoint configurations. 
* **Self-Contained Artifact Model:** Rather than separate-source architectures (which are prone to container environment mismatch), we bundle all weight checkpoints and serving code files inside a single `model.tar.gz` archive. SageMaker extracts the archive to `/opt/ml/model/` at runtime, mounting the serving script directly into the container's search path.

* **Resolution:**
  1. Patched the describe script with a **fail-fast conflict guard** that raises a `ValueError` if explicit S3 URLs are passed when config ARN is present.
